# Mon Site

Site web statique développé en HTML, CSS et JavaScript vanilla.

## Aperçu

> Décrivez brièvement votre site ici.

## Structure du projet

```
mon-site/
├── index.html        # Page principale
├── css/
│   └── style.css     # Styles globaux
├── js/
│   └── main.js       # Scripts JavaScript
├── assets/
│   └── images/       # Images du site
├── .gitignore
└── README.md
```

## Lancer le projet en local

Ouvrez simplement `index.html` dans votre navigateur, ou utilisez une extension comme **Live Server** (VS Code).
